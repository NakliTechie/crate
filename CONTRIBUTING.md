# Contributing to Crate

Crate is AGPL-3.0-or-later. Every new code file gets an SPDX header (`// SPDX-License-Identifier: AGPL-3.0-or-later` or HTML-comment equivalent). No GPL-2.0-only deps; no proprietary deps. Document vendored libraries in [`lib/vendor/LICENSES.md`](lib/vendor/LICENSES.md).

See [`crate-browser-handoff-v1.0.md`](../private-mesh/docs/specs/crate-browser-handoff-v1.0.md) (in the sibling `private-mesh` repo) for build conventions, dependency policy, and per-milestone gates.
